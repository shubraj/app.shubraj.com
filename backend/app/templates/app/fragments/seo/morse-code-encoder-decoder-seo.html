Morse Code Encoding and Decoding Made Simple

Convert any text to Morse code (dots and dashes) or decode Morse code back to readable text. This tool supports letters, numbers, and common punctuation, with audio playback to help you learn Morse code patterns.

What is Morse Code?

Morse code is a method of encoding text using sequences of dots (.) and dashes (-) to represent letters, numbers, and punctuation. Originally developed for telegraph communication, it's still used in radio communication, amateur radio, and as a learning tool.

How It Works

  • Encoding: Each character is converted to its corresponding Morse code sequence. Spaces between words become " / ".
  • Decoding: Morse code sequences are converted back to text by matching patterns to characters.
  • Audio Playback: The tool can play the Morse code audibly to help you learn the sound patterns.

Morse Code Basics

  • Dots (.) and Dashes (-): Dots are short signals, dashes are three times longer.
  • Spacing: Letters within words are separated by spaces, words are separated by " / ".
  • Case Insensitive: Input text is automatically converted to uppercase for encoding.
  • Common Characters: Supports A-Z, 0-9, and punctuation like period, comma, question mark.

Use Cases

  • Learning Morse code: Practice encoding and decoding to improve your skills.
  • Communication: Encode messages for transmission via radio or other methods.
  • Education: Teach students about historical communication methods.
  • Hobbies: Use in amateur radio, escape rooms, or puzzle games.

Tips for Learning

  • Start with common letters like E (.), T (-), and A (.-).
  • Use the audio playback feature to associate sounds with patterns.
  • Practice with short words before moving to longer sentences.
  • Memorize the most common patterns first (E, T, A, N, O, I).
