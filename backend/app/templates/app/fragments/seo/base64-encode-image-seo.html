Turn Small Images into Base64 for Inlining

Inlining images as Base64 can simplify quick demos and single‑file prototypes. Drop an image here to generate a data URL you can paste into HTML, CSS, or JSON fixtures.

When Inlining Makes Sense

  • Design previews: Share a minimal HTML file without external assets.
  • Component docs: Keep example code self‑contained and easy to copy.
  • Prototyping: Skip asset pipelines while exploring ideas.

Trade‑offs

  • Base64 increases size; prefer normal image files in production.
  • Long data URLs can be unwieldy in code reviews and diffs.
  • CDNs cache images better than inline strings for real apps.
